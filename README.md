# Data-Storytelling-Statistical-Validation
Data Storytelling &amp; Statistical Validation focused on analyzing sales data to uncover business insights using Python and Power BI. Performed data cleaning, exploratory data analysis, dashboard creation, and statistical tests (Correlation, T-Test, Chi-Square) to validate findings and provide data-driven business recommendations.

# Project Overview
This project focuses on analyzing sales data to uncover meaningful business insights through Data Storytelling and validate those insights using Statistical Analysis. The project demonstrates how data-driven techniques can help organizations make informed business decisions.
The analysis includes data exploration, visualization, statistical testing, business insights, recommendations, and an interactive Power BI dashboard.

# Objectives
- Analyze sales performance using Python and Power BI.
- Identify trends in sales, profit, and customer behavior.
- Perform statistical validation on business insights.
- Build an interactive dashboard for decision-making.
- Present findings through data storytelling.

*Dataset Used:** Sales Dataset (cleaned_sales_data.csv)
